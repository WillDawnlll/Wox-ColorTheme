![sample2](sample2.png)

# Install
Just copy `ColorTheme.xaml` and `src` to the Wox theme directory.

For example:
`C:\Users\UserName\AppData\Local\Wox\app-1.3.524\Themes`

Then choose it in your Wox settings.
